# Android 内存优化

这个目录包含了Android内存优化相关的资料、工具和示例代码。

## 主要内容

### 内存泄漏检测

- LeakCanary使用示例
- 常见内存泄漏场景和解决方案
- 自定义内存泄漏监控工具

### 内存抖动优化

- 对象池实现示例
- 避免频繁GC的最佳实践
- 性能监控示例代码

### 图片内存优化

- 高效加载大图片
- 图片缓存策略
- 图片压缩技术
